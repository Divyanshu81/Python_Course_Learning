WEATHER API RETURNS TEMPERATURE

This website return average temperature of the day of following stations in the given request format

URL format : 127.0.0.1:5000/api/station/date (yyyy-mm-dd)

For particular date and station

URL Example :127.0.0.1:5000/api/10/1988-10-25

For particular one station

URL Example :127.0.0.1:5000/api/10

For particular Year

URL Example :127.0.0.1:5000/api/10/yearly/1988
